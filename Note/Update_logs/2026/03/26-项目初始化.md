## 实现功能

### 1. MDConverter 类
- 实现 `__init__(encoding, css_file)` 构造函数
- 实现 `to_html(input_path, output_path, wrap_html)` 转换方法
- 支持自定义文件编码（默认 UTF-8）
- 支持自定义 CSS 样式文件
- 支持 wrap_html 参数控制输出格式

### 2. MarkdownParser 类
- 实现 `parse_file(file_path)` 方法，从文件路径解析
- 实现 `parse_content(content)` 方法，直接解析内容字符串
- 实现 `_extract_title(content)` 方法，提取第一个一级标题
- 实现 `_extract_metadata(content)` 方法，提取 YAML Front Matter 元数据

### 3. HTMLExporter 类
- 实现 `export(parsed_data, output_path)` 方法，导出到文件
- 实现 `export_to_string(parsed_data)` 方法，返回 HTML 字符串
- 实现 `_wrap_html(body_content, title)` 方法，包装完整 HTML 文档
- 实现 `_get_custom_css()` 方法，读取自定义 CSS 文件
- 实现 `_markdown_to_html(markdown_content)` 函数，使用 markdown2 转换
- 自动创建输出目录（如果不存在）
- 内置完整的 CSS 样式，支持中文显示

### 4. markdown2 扩展支持
启用以下扩展功能：
- fenced-code-blocks（围栏代码块）
- tables（表格）
- toc（目录）
- code-friendly（代码友好）
- footnotes（脚注）
- strike（删除线）
- task_list（任务列表）
